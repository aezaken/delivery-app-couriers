# Courier App

Приложение для курьеров службы доставки. Позволяет отслеживать заказы, видеть маршрут доставки, управлять своим статусом и получать уведомления.

## Основные функции

- **Аутентификация:** Вход в систему с проверкой учетных данных.
- **Панель управления:** Главный экран с вкладками заказов, баланса, чата и настроек.
- **Управление заказами:** Просмотр активных заказов, изменение их статуса (в пути, доставлен и т.д.).
- **Геолокация:** Отслеживание местоположения курьера в реальном времени и отправка координат на сервер.
- **Уведомления:** Push-уведомления о новых заказах и изменениях статуса.
- **Баланс:** Просмотр текущего баланса и истории операций.
- **Чат:** Общение с диспетчером.

## Архитектура и технологии

Приложение построено на Flutter с использованием архитектуры Provider для управления состоянием.

### Основные зависимости

- **Provider:** Для управления глобальным состоянием приложения.
- **Firebase:** `firebase_core` и `firebase_messaging` для аутентификации и push-уведомлений.
- **Геолокация:** `geolocator` для получения текущего местоположения.
- **Хранение данных:** `shared_preferences` для локального хранения данных (например, токена).
- **Сетевые запросы:** `dio` для взаимодействия с API бэкенда.
- **Утилиты:** `intl` для локализации, `flutter_dotenv` для загрузки конфигурации.

### Структура проекта

```
lib/
├── main.dart            # Точка входа, инициализация приложения
├── pages/              # Экраны приложения
│   ├── auth/           # Экран авторизации
│   ├── balance/        # Экраны баланса
│   ├── chat/           # Экран чата
│   ├── learning/       # Обучающий экран
│   └── menu/           # Главное меню и вкладки
├── providers/          # Классы для управления состоянием
├── services/           # Сервисы для работы с API, уведомлениями и т.д.
├── utils/              # Вспомогательные утилиты
└── widgets/            # Повторно используемые виджеты
```

### Интеграция с бэкендом

Приложение взаимодействует с бэкендом на Node.js через REST API. Все сетевые запросы обрабатываются через `ApiService` из общего пакета `shared_data`.

### Push-уведомления

Используется Firebase Cloud Messaging. Приложение обрабатывает уведомления как в активном состоянии, так и в фоне/на закрытом экране блокировки.

## Запуск приложения

1. Убедитесь, что установлен Flutter SDK.
2. Установите зависимости:
   ```bash
   flutter pub get
   ```
3. Запустите приложение:
   ```bash
   flutter run
   ```

Приложение автоматически загрузит конфигурацию из `.env` файла и инициализирует необходимые сервисы.
